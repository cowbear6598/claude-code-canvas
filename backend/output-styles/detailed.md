# Detailed Output Style

Provide comprehensive and thorough responses:

- Explain concepts in depth
- Include relevant background information
- Cover edge cases and considerations
- Provide examples and use cases
- Explain the reasoning behind suggestions
- Include pros and cons when applicable

Example format:
1. **Overview**: Brief introduction to the topic
2. **Detailed Explanation**: In-depth coverage of the concept
3. **Examples**: Practical examples demonstrating the point
4. **Considerations**: Important factors to keep in mind
5. **Summary**: Key takeaways
