# 多人游標共享 User Flow

## 情境一：加入已有他人在線的 Canvas

- 使用者 A 開啟 Canvas，畫面上沒有其他游標
- 使用者 B 開啟同一個 Canvas
- A 的畫面上出現一個彩色箭頭游標，代表 B 的位置
- B 的畫面上出現一個彩色箭頭游標，代表 A 的位置
- 每個連線者的游標顏色不同，用顏色區分彼此

## 情境二：移動滑鼠，其他人看到游標即時更新

- 使用者 A 在 Canvas 上移動滑鼠
- 使用者 B 的畫面上，A 的游標位置跟著更新（約每 100ms 一次）
- 游標位置以 Canvas 座標呈現，不會因為各自的縮放或位移而跑掉

## 情境三：使用者離線，游標消失

- 使用者 A 關閉頁面或網路中斷
- 使用者 B 的畫面上，A 的游標立即消失
- 不留下任何殘影或停滯游標

## 情境四：使用者切換到另一個 Canvas

- 使用者 A 從 Canvas X 切換到 Canvas Y
- Canvas X 上的其他人看到 A 的游標消失
- Canvas Y 上的人看到 A 的游標出現
- A 看不到 Canvas X 的人的游標，只看到 Canvas Y 的人的游標

## 情境五：三人以上同時在線

- 使用者 A、B、C 同時在同一個 Canvas
- 每人的游標顏色不同，可以在畫面上同時看到另外兩人的游標
- 任何一人移動，其他兩人都會看到該游標更新
- 任何一人離線，其他兩人畫面上該游標消失
